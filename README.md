# GolabiJs

Requirements
------------
features requires the following to run:

  * [jQuery][jQuery] 1.0+


[jQuery]: https://jQuery.com/

Usage
-----

to put element in the center of page:
```js
$('#elm').center();
```

to put element in the center of a box element:
```js
$('#elm').center('#box');
```
